# 🚀 Overview

**Pravarthana HRMS** is a modern full-stack HR management platform designed to help companies manage employees, attendance, communication, and HR workflows efficiently.

The system combines:

• A **public marketing website**  
• A **complete internal HRMS application**

---

# 🌐 Features

## Public Website

- Modern landing page
- Product showcase & features
- Pricing plans
- Contact / demo request forms
- Fully responsive design

---

## HRMS Application

### Employee Management
Manage the complete employee lifecycle including onboarding and employee profiles.

### Attendance Tracking
Manual check-in/check-out system with attendance reports.

### Leave Management
Employees can apply for leave and managers can approve or reject requests.

### Offer Letter Generator
Generate professional PDF offer letters automatically.

### Real-time Chat
Internal messaging system for communication between employees.

### Role-Based Access Control
User roles supported:

- Super Admin
- HR Admin
- Manager
- Employee

# 🛠️ Tech Stack

## Frontend

- **Framework:** Next.js 14 (App Router)
- **Library:** React 18
- **Styling:** Tailwind CSS + shadcn/ui
- **State Management:** Zustand / React Context
- **HTTP Client:** Axios
- **Realtime Communication:** Socket.IO Client

---

## Backend

- **Framework:** Spring Boot 3
- **Language:** Java 17
- **Database:** PostgreSQL
- **Authentication:** JWT + Spring Security
- **WebSocket:** Spring WebSocket (STOMP)
- **PDF Generation:** iText / Apache PDFBox
